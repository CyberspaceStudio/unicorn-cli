# unicorn-cli

unicorn: 独角兽 （ 🦄） 

## Introduction
青鸢工作室前端开发脚手架

### Features
现存web，miniapp两套模板，分开维护。

## Requirements

Windows用户需要在管理员模式下进行全局安装

## Installation
```node
    npm install -g unicorn-cli
    unicorn-cli create app-name

```
